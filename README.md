## Installation

```bash
$ npx -p @nestjs/cli nest new NestJS-Learn
$ yarn add @nestjs/graphql apollo-server-express graphql-tools graphql
```

or

```bash
$ git clone https://rchagas@bitbucket.org/rchagas/nestjs-learn.git
```

## Dependencies

```bash
# npm
$ npm install -g npx

# yarn (for clone)
$ yarn install

# nodemailer
$ npm install nodemailer

# rxjs
$ npm install rxjs

# moment
$ yarn add moment
```

## Running the app

```bash
# development
$ yarn start:dev

# production
$ yarn start
```
